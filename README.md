# The Beast Portfolio



Welcome to beastPortfolio! This is a **modified version** of the Master Portfolio project with some additional features.

## Features 🚀

✔ Clean, responsive, and customizable design

✔ Showcases skills, projects, and experience

✔ Open-source projects linked with GitHub

✔ Integrated chatbot (Botpress)

✔ Dynamic theme switching (Every **5 clicks**, a new theme is applied)

## Getting Started 📌

To run this portfolio locally:

```bash
# Clone the repo
git clone https://github.com/DebugBeast/beastPortfolio.git
cd beastPortfolio

# Install dependencies
npm install

# Start the development server
npm start
```

## Customization ✨

Edit `src/portfolio.js` to update your details like skills, experience, and projects. Themes can be modified in `src/theme.js`.


## Deployment 🚀

Easily deploy via GitHub Pages, vercel or Netlify:

```bash
npm run build
# Deploy using GitHub Pages
git init
git add .
git commit -m "Deploy"
git push -u origin master
```

## Tech Stack 🛠

- **React.js**
- **JavaScript**
- **Styled Components**
- **Botpress** (for chatbot integration)
